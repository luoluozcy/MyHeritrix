活动赞助品牌

总有一款你最钟情的Little thing。

 

也许是你寻觅已久的上世纪复古耳环，也许是你re-design的手作刺绣口金包，也许是外祖母的古董手表，也许是朋友淘来的蓝宝石胸针，也许是一顶英伦呢绒画家帽，也许是一副经典雅致的镜框，也许是精巧的黄铜白水晶戒指，也许是斑斓的乔其缎印花丝巾。俏皮时尚的小物们经过巧妙组合，赋予搭配别样的灵动气息，细微之处的独特魅力自然而成。

 

亲爱的iStyler，你们在世界的各个角落，用什么样的姿态，在做什么美好的事情？这一季，Little thing恋物志与iStyle联合举办“恋物搭配活动”，以细节突显个性，诠释不一样的穿衣风格。

 

我们期待，看到你们的模样，给我们惊喜，分享恋物之美好。

活动规则：

1. 上传全身穿搭的同时，至少有一张细节图展示搭配小物（如包包、帽子、饰品等）。

2. 描述搭配中你最喜欢的Little thing，不要吝啬文字哦。

3. 如有一件或多件单品来自Little thing，请标明。

活动时间：

11月22日  —  12月17日

奖项设置：

1. 风格恋物奖2名《恋物志》2013年全年杂志 + 4周年纪念版礼盒

2. 创意恋物奖2名《恋物志》2013年全年杂志 + 4周年珍藏版丝巾

3. 意趣恋物奖2名4周年珍藏版丝巾裙

4. 人气恋物奖2名Little thing环保牛仔手工鞋

所有获奖女生都可获得Little thing 复古设计印花连身袜一双，优秀搭配作品将有机会刊登于Little thing
杂志二月刊

评选标准：

最终评奖结果由iStyle网站与Little thing杂志共同评选

以细节突显个性，诠释不一样的穿衣风格。iStyle联合Little thing举办恋物搭配活动，共收到354个搭配作品。活动圆满结束，感谢各位iStyler的热心参与。

风格恋物奖：模样公主、小娟

创意恋物奖：白-白、达子

意趣恋物奖：张宇莹-小花、DadA-NaKo

人气恋物奖：NumberW、差很多先森。
