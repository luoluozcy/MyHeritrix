从电影到秀场，从引领风尚的明星到我们身边的It Girl，怀旧复古风潮从未销声匿迹，七十年代的松糕鞋，八十年代的大垫肩，九十年代的牛仔背心，这一切在今天看来依旧时髦！

 

所谓的Vintage，早已从过去的小众时尚走入了大众，前人们留下的那些魔法果实出现在二手市场，填补了现代人心中寻宝探险的渴望。带着历史特有气息与灵魂的物品，每一样似乎都有着说不完的故事，这正是vintage的魅力所在！爱上vintage，因为它是旧的，因为留在它身上的那份古味，因为它可以被再次使用的美好！

 

那么，不管你在世界的哪一端，让我们抛开fast fashion，感受vintage无与伦比的美丽！

活动规则：

1. 有一件或者多件单品为vintage，请标明。

2. 为搭配用到的vintage单品拍细节图，让大家更好地欣赏它的美。

3. 在搭 配描述里，说说你获得这件单品的来历，告诉我们有关它的故事，或是喜欢它的原因。

活动结束后会从喜欢数较高的热门搭配中评选出若干奖项，并给予奖品。

活动时间：

1月4日-2月4日

奖项及评选规则：

1. 最佳vintage风格搭配奖 x 2

2. 创意vintage单品搭配奖 x 3

3. vintage搭配入围奖 x 4

TOP30作品入围评选，最终评价结果将由造型师余久兴和ISTYLE网站共同投票评选。
