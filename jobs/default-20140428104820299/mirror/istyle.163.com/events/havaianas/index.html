活动赞助

热火朝天的夏季，我们迎来沙滩、海浪、树荫与绿意。褪去多余的衣裳，恣意享受闲适与清凉。

无论你与T恤牛仔短裤组合难舍难分，还是情迷印花波卡圆点无法自拔，舒适惬意的清凉穿搭都是正当红的
夏日主角。

如何用装扮点亮夏天？棉柔短裙mix人字拖或者穿出一身缤纷乐 ，无穷想象的碰撞带来妙趣横生的火花。

这一季，iStyle携手时尚品牌Havaianas举办这场斑斓的清凉一夏主题趴。

抛开条条框框的束缚，show出你最正的夏日清凉style，大声宣告夏天的自由与轻快吧。

活动规则

1. 上传符合主题的全身搭配。

2. 如有单品来自Havaianas，请标明。

活动时间

8月2日  —  8月25日

奖项设置

1. 最佳穿搭奖 X 11000元代金券

2. 意趣风格奖 X 3500元代金券

3. 人气搭配奖 X 3500元代金券

评选标准

最终评奖结果将由iStyle与Havaianas品牌共同评选
