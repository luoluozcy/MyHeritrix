【日本冈山】仓敷：樱花季的柔软时光（上）   

2014-04-18 09:27:00|  分类： j调de华丽,日本, |  标签： |举报 |字号大中小 订阅

 

这里的老城区已有300多年历史，这里的街道诉说着昔日的繁华，这里的美术馆收藏了莫奈、毕加索等大师的艺术作品，这里的樱花季唯美静谧……这里是仓敷，它惊艳了岁月，温柔了时光。位于日本本州西部冈山县的仓敷市在日本国内的问卷调查中被选为日本第一的商屋街，仓敷的迷人之处在于，这里四分之一的建筑都建在运河两岸，古老的黑色砖瓦建成的仓库被改成博物馆，成为仓敷的主要特色之一。300多年前，这一带是幕府的直辖领地，至今仍保存着从日本有武士的江户时代到明治、大正时代的街区。因为有一家名声斐然的私人博物馆――大原美术馆躲过了战争，所以仓敷完整地保留着很多日本自古以来的白墙建筑和古西洋文化建筑。

 

 

 

这次在仓敷住的旅馆我非常喜欢，叫仓敷日式旅馆（Ryokan Kurashiki），总共只有5间房，是仓敷地区最好的日式旅馆，从旅馆就能俯瞰运河两岸风光。这一天我们从东京坐新干线到冈山再转到仓敷已是晚上，旅馆的服务人员还在等我们，办理入住手续时，为我们提供了清新的绿茶和可口的当地甜点。预定房间要通过旅馆的官网上预定：http://www.ryokan-kurashiki.jp，这家旅馆在当地非常有名，本身就是一栋几百年的历史老建筑。

 

不得不说我非常佩服这家只有5间房间的日式旅馆，所有的插花摆设都非常精致，精湛的插花技巧居然还能体现在美食料理上，请继续往下看。

 

这是旅馆的大厅，办理入住手续的地方，其实这里也是以前这户人家的客厅，木质的桌子和椅子表面磨得发亮，这些都是岁月留下的痕迹。

 

客厅一角，在这里可以自己泡茶泡咖啡，这里是我每天吃完早餐必来报道的地方。

 

旅馆只有5间房但不代表是一家小旅馆，长长幽深的走廊充满了典雅的日式范儿，轻步前行心情也会变得好起来。

 

 

在走廊的一角有一处迷你的日式庭院，旁边围着一圈榻榻米，果然是细节决定成败啊。

 

这家的插花不仅仅是造型好看，关键是能和周围的环境空间配合的相得益彰天衣无缝。


客厅外面是一个精致典雅的日式庭院，恬静、含蓄、淡然，夜里下着小雨，雨润无声。

 

镂空花纹吊灯里透出的暖光在雨夜里显得格外清幽朦胧。

 

我们的房间是复式的，这个角度是站在两楼拍的。

 

二楼的小阁楼还保有300多年前的样子，从来不曾改变过。

移开一楼客厅的门，古朴典雅的日式范儿扑面而来。

 

卧室的床非常大，所以说我们住的这间复式最多是可以睡六个人的，这里睡两人，两人睡客厅，还有两人睡两人阁楼。

 第二天去餐厅用早餐，还未坐定差点惊呼出声，要不要这么像一幅画啊，太舒心惬意了。

 

精致的日式早餐，赏心悦目，给我一天的正能量。


 

烤鱼都能用这么文艺的炉子，服了...

 


 

吃完早餐去客厅喝咖啡，这是我对这座旅馆公共区域最喜欢的一个角落，如果哪里也不去，我估计在这里都能懒上一天。

 

 

仓敷旅馆的女将中村小姐送我们出门，是个大美女哦~~~笑起来特别甜美：）


 

吃完早餐有力气开始走街串巷。

 

仓敷美观地区至今还保留着当时富商大贾的仓库和宅第，流传着近代商人的荣华和文化，整个街市独具特色，别有情趣。

 

眺望远处，山头上一片姹紫嫣红，樱花时节的仓敷无疑是最美的。

 


仓敷的大原美术馆大大出乎了我的意料，这里藏有罗丹、马蒂斯、毕加索、莫奈等艺术大师的代表作，这些都是原来的纺织业巨头大原先生的藏品，这里也是日本最早的私人西洋美术馆。


 

莫奈一生中创作过许多幅《睡莲》，这是其中的一幅。

 

去完美术馆回旅馆吃午餐，午间定食是1850日元，折合人民币也才110元左右。

 

待在格子里的一个个小盘都非常可爱，吃的不是食物，是精致。



白天的庭院掩不住的春色满园。

 

黄昏的街道被夕阳染成金黄，温情在街头巷尾肆虐。

 

晚餐，我们在餐桌上“赏樱”。

 

 

等刺身拼盘上来的时候，我彻底被料理长制服了。这我大概是我见过的最美的生鱼片拼盘了，把日式插花与料理的精髓糅合的天衣无缝，美艳的不忍心吃。

 

无论从哪个角度欣赏都是在小心脏上重重的震撼了一把，美成这样是不是有点过分了？

 

 

料理长用萝卜制作了一片“樱花花瓣”和插在碎冰上的樱枝完美的融合在一起，看起来更像是一件艺术品。

 

芥末是现磨的。

  

最后的点心是樱饼，非常应景。

 

吃完晚餐我们步行10分钟去原来旅馆主人儿子的bar喝了一杯，主人非常喜欢旅行，这里很多都是他从世界各地带回来的古董，详细地址可询问旅馆工作人员，非常神秘的一个地方。

 

晚上的仓敷地区静谧祥和，和白天游人如织完全不一样，仿佛脱胎换骨成了一个与世隔绝的小镇。


左边就是仓敷日式旅馆，几百年来它如斯伫立在运河边，在夜色中散发着迷人的气质。
