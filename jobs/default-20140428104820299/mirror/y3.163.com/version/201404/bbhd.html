活动时间：4月23日维护后至4月30日维护前

活动规则：

>> 通关任意PVE副本（10点以上活力消耗），有50%几率可以获得1个冢虎秘藏
>> 在一场非双人匹配的PVP（匹配魏蜀争霸、逐鹿中原、三国大战、排位赛）战斗中失败，有50%几率可以获得1个冢虎秘藏
>> 在一场非双人匹配的PVP（匹配魏蜀争霸、逐鹿中原、三国大战、排位赛）战斗获得胜利，可以获得1个冢虎秘藏
>> 双人匹配竞技图（匹配魏蜀争霸、逐鹿中原、三国大战、排位赛）战斗，无论胜负，可以获得1个冢虎秘藏
>> 每人每天最多可以获得2个冢虎秘藏

活动时间：4月30日维护后至5月14日维护前

参与条件：8级以上的玩家可以参与

阵营划分

>> 玩家将分为三个阵营：司马懿、诸葛亮和周瑜。阵营一旦选中，无法更改。
>> 界面打开时，默认选择的是当前人数最少的阵营。不显示各阵营当前总积分，只显示人数比例。阵营人数比例超过≥34%，则无法加入该阵营

阵营胜负

>> 活动结束时，每个城镇中阵营军师剩余血量最多的一个阵营，可以占领该城镇血量。
>> 每日活动结束时，根据本日占领城镇数量不同发放奖励。
>> 每日活动开始时，会重置阵营军师的血量。
　

阵营战斗

>> 活动期间，会在所有三国城镇出现分别代表三个阵营的代表军师，点击可以挑战敌对阵营军师，挑战成功后可以降低阵营军师血量。
>> NPC有三个选项，分别对应三个副本

斩将PVP副本

>> 建议装备分数低于6W可以参与，PVE装备不生效。
>> 玩家各自拥有10000金币，可以购买PVP竞技装备来通关副本。
>> 通关后会对敌方阵营军师造成伤害。

杀敌PVE副本

>> 建议装备分数在6W-10W间参与
>> 有难度较高的名将Boss
>> 通关后会对敌方阵营军师造成伤害。

军师之殁副本

>> 建议装备分数10W+参与。
>> 有难度高的军师BOSS。
>> 通关后会对敌方阵营军师造成大量伤害。
