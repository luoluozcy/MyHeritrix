礼满乾坤

活动时间：4月30日维护后至5月14日维护后

一马当先

>> 七日登录可以获得贾诩。
>> 新玩家完成七日登陆历练任务，可以获得七日历练好礼。
>> 七大英雄任选畅玩：新手前七日登录，即可在7大英雄中任选一个英雄体验畅玩7天。

重返沙场

>> 指定时间内回流的玩家，可以获得回流相关奖励。
>> 玩家上次登录是在14天以前，登录城镇即可获得回流礼包。
>> 一个玩家一个月最多只能领取一次礼包，月底刷新CD。
>> 组队经验加成：与其他玩家5人组队时，回流玩家和与回流玩家组队的队员均可获得10%的基础经验加成。
>> 回流玩家可以获得特殊称谓：重返沙场，该称谓持续两周
>> 可自选PVP方向或PVE方向礼包，更合心意
